## 3. Plot 2 — TFT-DCP Architecture Diagram

**Location:** Section 3 (Model Architecture), replace `[Plot 2]`.

**Type:** Programmatic architecture block diagram using matplotlib patches.

**Components to show:**
1. Input streams: Static features (route priors, airline, distance), Dynamic sequence (weather,
   ASPM, turnaround), Weather context
2. TCN Encoder (3 dilated causal conv blocks with LayerNorm) → H_dynamic
3. GRN (Gated Residual Network) for static embedding
4. Historical Retrieval Module (cosine similarity database lookup, top-k, softmax fusion)
5. MS-CA-EFM (Multi-Scale Channel Attention with Excitation-Feature Modulation)
6. Delay Propagation Network (learnable β chain model)
7. Output head → predicted DepDelay + risk_score

**Generated by:** `generate_report_figures.py` → `fig_architecture.png`
